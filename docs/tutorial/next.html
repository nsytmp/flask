继续发展！¶

在整个教程中，您已经了解了很多Flask和Python概念. 返回并查看教程，并将您的代码与您实现的步骤进行比较. 将您的项目与:gh:`example project <examples / tutorial>`进行比较，由于本教程的逐步特性，这可能看起来有点不同.

Flask比你到目前为止看到的要多得多. 即便如此，您现在已经准备好开始开发自己的Web应用程序了. 查看:ref:quickstart，了解Flask可以做些什么，然后深入了解文档以继续学习. Flask在幕后使用`Jinja`_，Click，Werkzeug`_和`ItsDangerous，它们都有自己的文档. 您还会对:ref:`extensions`感兴趣，它们可以使用数据库或验证表单数据等任务变得更容易，更强大.

如果您想继续开发Flaskr项目，可以参考以下内容:

  • 显示单个帖子的详细视图. 点击帖子的标题转到其页面.

  • 喜欢/不像帖子.

  • 评论.

  • 标签. 单击标签会显示包含该标签的所有帖子.

  • 一个按名称过滤索引页面的搜索框.

  • 分页显示. 每页只显示5个帖子.

  • 上传图片以配合帖子.

  • 使用Markdown格式化帖子.

  • 新帖子的RSS提要.

玩得开心，制作出色的应用程序！

It basically means: do whatever you want with it as long as the copyright in Flask sticks around, the conditions are not modified and the disclaimer is present. 作者¶

Flask由Pallets团队和社区贡献者开发和维护. 它由Armin Ronacher创建. 核心维护者是:

  • 大卫勋爵(大卫)

  • Adrian Mönnich (ThiefMaster)

  • Armin Ronacher (mitsuhiko)

  • Marcus Unterwaditzer (untitaker)

完整的贡献者列表可以从git获得:

git shortlog -sne

一般许可证定义¶

以下部分包含Flask的完整许可文本和文档.

  • “作者”在此指的是:ref:`authors`部分中列出的所有作者.

  • The “Flask许可证” applies to all the source code shipped as part of Flask (Flask itself as well as the examples and the unittests) as well as documentation.

  • “Flask 艺术品许可证 ”适用于该项目的Horn-Logo.
